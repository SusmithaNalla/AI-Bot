# AI Topic Explainer

AI Topic Explainer is a web application that helps users understand study topics quickly using AI.

The user enters a topic and the application generates a simple explanation using the Gemini API.

## Features
- Enter any study topic
- AI generates easy-to-understand explanations
- Clean and modern UI
- Example topic suggestions
- Built with modern web technologies

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Gemini API
- Vercel (Deployment)

## Installation

Clone the repository:

git clone https://github.com/Sathvika2105/ai-topic-explainer.git

Install dependencies:

npm install

Run the development server:

npm run dev

Open http://localhost:3000 in your browser.
